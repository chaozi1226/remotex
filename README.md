# RemoteX

RemoteX 是一个聚焦远程协作的社区。

当前，我们关注的话题有：

1. **外包** : 如何改变传统的软件开发怪圈
2. **兼职** : 寻求灵活的雇佣方式
3. **远程** : 探讨远程工作新模式
4. **众包** : 实践新的需求解决方法论
